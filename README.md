

# Instrucciones para ejecutar la aplicación


## Ejecutar localmente

**Prerequisitos:**  Node.js


1. Instalar dependencias, ejecutar el siguiente comando desde la terminal:
   `npm install`
2. Lanzar la app: ejecutar el siguiente comando:
   `npm run dev`
3. Pasar el mouse por los items del menú y hacer click sobre los submenús para navegar en el sitio
